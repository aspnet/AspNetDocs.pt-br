# <a name="owin-and-katana"></a>OWIN e Katana

O Katana é um conjunto flexível de componentes para criar e hospedar aplicativos Web baseados em OWIN (Open Web interface for .NET). O novo desenvolvimento deve usar [ASP.NET Core](/aspnet/core/introduction-to-aspnet-core). As `Microsoft.Owin.*` bibliotecas não estão disponíveis para o .NET Core porque ASP.NET Core tem substituições equivalentes para elas.
